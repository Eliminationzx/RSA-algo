# ![logo](http://s32.postimg.org/ga6dwr6fl/icon.png) RSA-algo

## Introduction
Application for en/de-cryption with RSA method.

Basic functions:
* Text en/de-cryption
* File en/de-cryption
* Export keys
* Import keys
* Logging
* Setting

## Requirements
* OS Windows 7/8/10
* 3 - 20 MB free space
